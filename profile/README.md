## 🚀 Why Choose Snigdha OS?

**Snigdha OS** is a security-focused, lightweight Linux distribution built on Arch Linux, designed for developers, cybersecurity professionals, and privacy-conscious users. Featuring the latest Linux Zen and LTS kernels, we deliver:

- ⚡ **Blazing Performance**: Optimized kernel and minimal resource usage
- 🛡️ **Security First**: Built-in hardening and privacy tools
- 🧩 **Customization**: Modular Blackbox Framework for workflow optimization
- 📦 **Arch Compatibility**: Access to AUR and latest packages
- 🔄 **Rolling Releases**: Continuous updates with stable base

## 🛠️ Getting Started

### System Requirements
- **Processor**: x86_64 compatible
- **RAM**: 2GB+ (4GB recommended)
- **Storage**: 20GB+ available space

### Quick Installation
```bash
curl -sSL https://install.snigdhaos.org | bash
```

[Full Installation Guide](https://docs.snigdhaos.org/installation) | [Hardware Compatibility List](https://wiki.snigdhaos.org/hcl)
